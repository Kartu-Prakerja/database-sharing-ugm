Hai Alisa Putri, halaman ini hanya dapat diakses selama 7 hari dimulai dari waktu permintaan akses. Untuk mendapatkan akses kembali, silakan kirim permintaan ulang untuk mengunduh database.

Data dan Dokumentasi

Harap mendownload dan membaca dokumen “Guidebook” terlebih dahulu sebelum menggunakan dataset. Dalam dokumen tersebut, dijelaskan mengenai regulasi, skema program, karakteristik penerima Prakerja, metode sampling, jumlah sampel, codebook, hingga penjelasan cara menggunakan database yang berbeda-beda untuk masing-masing tahun. 

Pengguna perlu memahami konteks program dan user journey penerima Prakerja sebelum melakukan analisis terhadap database Prakerja. Sehingga analisis dapat dilakukan berdasarkan pemilihan variabel yang tepat, penggunaan metodologi analisis yang sesuai, dan interpretasi hasil yang akurat.

Database Prakerja dipisahkan berdasarkan tahun yaitu 2020, 2021, dan 2022 dan masing-masing tahun berisi dataset user journey, dataset survei evaluasi, dan dataset pelatihan.
