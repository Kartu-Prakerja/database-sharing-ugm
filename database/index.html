Unduh Database

Penjelasan Umum

Database Publik Prakerja bertujuan memberikan kesempatan untuk mendokumentasikan, mempelajari, dan menganalisis program Prakerja oleh siapa saja, termasuk peneliti, mahasiswa, akademisi, serta pembuat kebijakan di bidang ketenagakerjaan, pembangunan ekonomi, atau bidang lain yang relevan.  Database ini berisi data sampel penerima pada periode skema semi-bansos tahun 2020-2022. Data sampel Prakerja bersumber dari data User Journey dan Survei Evaluasi yang dilaksanakan oleh Manajemen Pelaksana Program Kartu Prakerja (MPPKP). Ketersediaan database ini diharapkan dapat mendorong penelitian dan diskusi yang mendalam untuk meningkatkan program Prakerja dan kebijakan publik Indonesia di masa depan.

Syarat dan Ketentuan Pengunduhan Database

Terima kasih atas minat Anda terhadap Database Publik Prakerja. Sebelum Anda mengakses Database Publik Prakerja, hanya menyetujui seluruh ketentuan dibawah ini:

  • Anda harus mendaftar dan mengisi informasi pada formulir pengunduhan database dengan benar dan jujur. 
  • Anda setuju untuk tidak mencoba menentukan identitas individu dan lembaga dari dataset ini, kecuali berkaitan dengan kode anonim yang telah disediakan dalam Database Publik Prakerja. Anda perlu menggunakan data sesuai dengan term and condition yang berlaku, termasuk menghormati anonimitas semua responden data. 
  • Anda perlu menjaga kerahasiaan dan keamanan dataset ini dengan tidak menyebarkan database publik Prakerja yang telah Anda download. Jika Anda bekerja bersama orang lain menggunakan data ini, Anda akan menganjurkan untuk mendaftar atau mendaftarkannya sendiri.
  • Anda setuju untuk mengirimkan kutipan dari setiap makalah, artikel, buku, atau publikasi lainnya yang menggunakan Database Publik Prakerja kepada kami melalui email xxxx.xxx@xxx.com , termasuk URL dari publikasi tersebut. 
  • Dalam semua laporan dan publikasi yang didasarkan pada Database Publik Prakerja, Anda wajib mengakui Database Publik Prakerja sebagai sumber data. Kutipan yang tepat dapat ditemukan di halaman pengunduhan Database Publik Prakerja.
